# Markdown Live Preview

[![md.airat.top](https://raw.githubusercontent.com/AiratTop/md.airat.top/main/canvas.png)](https://md.airat.top/)

A local, private markdown live preview tool with a split editor and preview. Runs as a pure static site and works on Cloudflare Pages or GitHub Pages.

The app is published and available at: https://md.airat.top/

## Features

- Live markdown rendering with GitHub-flavored markdown.
- Split layout: editor on the left, preview on the right.
- Sync scroll, reset, and copy actions.
- Dark mode based on browser settings with manual override.
- No server, no third-party services.

## Local development

Serve the folder with any static server:

```bash
python -m http.server
```

Then open `http://localhost:8000`.
